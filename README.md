# Omni-Mind - العقل الموحد

نظام متكامل لدمج أربعة نماذج ذكاء اصطناعي في واجهة واحدة.

## المميزات

- دمج أربعة نماذج ذكاء اصطناعي: DeepSeek، Gemini، Claude، ChatGPT
- نظام توجيه ذكي لاختيار النموذج المناسب تلقائياً
- واجهة مستخدم عربية سهلة الاستخدام
- إمكانية اختيار النموذج يدوياً
- عرض حالة كل نموذج

## متطلبات التشغيل

1. Node.js إصدار 16 أو أعلى
2. مفاتيح API للخدمات الأربعة
